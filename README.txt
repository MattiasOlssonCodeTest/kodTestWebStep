The solution is built in Visual Studio 2022 targeting .NET 7.0. 
To run the application you first need to install the database.
Open the solution in visual studio and open the "SQL Server Object Explorer".
Expand "SQL Server", right click on "(localdb)\MSSQLLocalDB..." select New Query.
Paste in the code from the file CreateDB.sql and execute.
Now run the project from within visual studio.
